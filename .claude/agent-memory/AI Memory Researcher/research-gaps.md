# Research Gaps Identified (2026-03-18)

## Critical Gaps
1. **No Predictive Processing framework** — Thesis cites CLS + GWT but not PP/AIF (Friston/Clark), the most active cognitive architecture framework in 2025-2026
2. **No knowledge graph / relational memory** — Flat vector search only; Mem0g shows 26% accuracy boost with graph augmentation
3. **Emotional taxonomy theoretically dated** — 12-category scheme conflicts with Barrett's Theory of Constructed Emotion; should evolve toward dimensional (valence/arousal/dominance)

## Important Gaps
4. **No explicit forgetting theory** — Active vs. passive forgetting distinction not addressed; FNNs (ICAART 2026) provide a model
5. **No world model** — Self-model + user model exist, but no structured representation of user's external environment (Damasio's framework requires both)
6. **Comparison table outdated** — Claims "not implemented" for features competitors now have (ChatGPT year-long recall, Gemini Personal Intelligence, Mem0 graph memory)
7. **CLS sampling strategy unspecified** — Both whitepaper and inner-life.md flag this as critical but leave it as open question
8. **Letta skill learning not addressed** — More formal than AnimaOS's behavioral rules
9. **No multi-modal memory** — Text-only assumption; voice/ambient sensing implications unaddressed
10. **No third-party memory governance** — Privacy implications of storing info about people other than the user
